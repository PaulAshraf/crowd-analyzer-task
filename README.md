# ✨ Charts App for CrowdAnalyzer ✨

## Install Packages

```bash
npm i
npm i -g json-server
```

## Run Locally

```bash
npm start
json-server --watch db.json --port 300
```

## Testing with Cypress

```bash
npm run cypress:run
```
